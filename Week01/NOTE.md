##### 2.复杂度分析

###### 2.1 复杂度表示

> 大O表示法(Big O notation)

常见的七种时间复杂度

```
O(1) : 常数复杂度(Constant Complexity)
O(log n) : 对数复杂度(Logarithmic Complexity)
O(n) : 线性时间复杂度(Linear Complexity)
O(n^2) : N平方(N square Complexity)
O(n^3) : N立方(N cubic Complexity)
O(2^n) : 指数(Exponential Growth)
O(n!) : 阶乘(Factorial)
```
